Sidequest 1.b: Dynamic Theme System for Documentation

Objective

Enhance the documentation HTML with a dynamic theme selector and sci-fi animation previews, allowing users to instantly preview and apply different color schemes across the entire document.

What We Did

  • Added Theme Selector: Users can choose from multiple sci-fi themes (Matrix, Cyberpunk, Dune, Star Trek, etc.) using a dropdown menu.
  • Dynamic CSS Variables: Each theme sets a palette of CSS variables for background, text, accent, code, and box colors. The entire document updates instantly when a theme is selected.
  • Live Animation Preview: The animation preview area updates to match the selected theme, showing a themed animation card.
  • Seamless Integration: All content, including code blocks and boxes, adapts to the selected theme for a consistent sci-fi look and feel.
