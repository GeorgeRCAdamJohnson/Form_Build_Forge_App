Form Builder README

Overview

The Form Builder allows you to create custom form templates for Jira issues by selecting and arranging Jira fields.

Features

  • Select from available Jira fields
  • Drag and drop field reordering
  • Save and load form templates
  • Duplicate field validation
  • Template management

How to Use

  1. Enter a template name
  2. Click on available Jira fields to add them
  3. Use ↑↓ buttons to reorder fields
  4. Click "Save Template" to save
  5. Use "Load Template..." dropdown to load saved templates
