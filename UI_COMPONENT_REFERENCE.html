Finding Components
1. Use browser DevTools to inspect element and find CSS class
2. Search for class name in styles.css
3. Search for component usage in .jsx files
4. Check all theme variants (.theme-modern, .theme-scifi, .theme-barbie)
Common Issues
‚Ä¢ Text hard to read: Check contrast colors in theme variables
‚Ä¢ Styling not applying: Ensure theme class is on body element
‚Ä¢ Drag not working: Check @dnd-kit imports and handlers
‚Ä¢ Theme not switching: Verify ThemeProvider wraps component
